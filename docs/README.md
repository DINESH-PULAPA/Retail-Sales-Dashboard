📊 Retail Sales Data Analysis (Excel Project)
📌 Project Overview

This project analyzes a Retail Sales dataset (1000 transactions) using Excel.
The goal is to practice data cleaning, analysis, pivot tables, and business insights—Day 1 of the 60-Day Data Analyst Roadmap.

📂 Files in This Project
File Name	Description
Retail_Sales_Analysis.xlsx	Main Excel file containing cleaning, summary sheet, pivot tables & charts
Retail_Sales_Data.csv	Raw dataset downloaded from Kaggle
🧹 1. Data Cleaning Steps

The raw dataset contained 9 columns:

Transaction ID
Date
Customer
Gender
Age
Product Category
Quantity
Price per Unit
Total Amount

Cleaning performed:

✔️ Removed duplicates

No duplicate rows found using Remove Duplicates.

✔️ Corrected data types
Date → Date format
Price, Quantity, Total Amount → Number format
Gender, Product Category → Text
✔️ Checked for missing values

No blank rows or missing values found.

✔️ Created new calculated column

Total Amount = Quantity × Price per Unit

📈 3. Key Insights (Very Important)
🛍 Product Category Insights
Electronics is the top-performing category (₹156,905).
Clothing is close behind, indicating strong demand.
Beauty has the lowest sales but still contributes significantly.
👥 Customer Demographics
Females contribute slightly higher sales (₹232,840) than males.
Average customer age is 41 years, meaning most customers fall in the working-age group (ages 30–50).
📌 Business Implications
Electronics and Clothing should be prioritized for marketing.
Female-focused promotions may increase sales further.
Beauty category can be targeted with discounts or bundle offers.
